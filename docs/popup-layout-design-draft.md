# 擴充 popup 視窗版面優化 — 草稿與評估

## 目前狀態（已做）

- **視窗尺寸**：popup 固定 400×560px，內容不超出邊框、不捲動。
- **緊湊模式**：在 iframe（擴充內）時自動啟用：標題列／資訊列／角色區／數值／按鈕列皆縮小間距與字級，角色區 min-height 降為 100px，一版內可看到深呼吸、番茄鐘、餵食、喝水、睡覺、散步、抱抱、天氣等。

---

## 外觀調整 — 可討論方向

### 1. 項目取捨（若仍覺得擠）

| 選項 | 作法 | 優點 | 缺點 |
|------|------|------|------|
| **A. 保留全部、僅縮小** | 維持現狀，只做視覺微調（圓角、對比、圖示大小） | 功能完整 | 資訊密度高，易顯得擠 |
| **B. 摺疊次要** | 深呼吸／番茄鐘／天氣收成「更多」或一顆圖示，點開再展開 | 主畫面更乾淨 | 多一層操作 |
| **C. 分頁** | 主畫面：角色 + 餵食/喝水/睡覺/玩/散步/抱抱；第二頁：深呼吸、番茄鐘、天氣、設定 | 每頁項目少、版面好排 | 要滑動或點頁籤切換 |
| **D. 只留最常用** | popup 只顯示：角色 + 餵食/喝水/睡覺/玩 + 陪伴模式；其餘移到「設定」或主站 | 版面最單純 | 部分功能要進主站或用設定 |

### 2. 視覺微調（不砍功能時）

- **按鈕**：目前 6 格（餵食/喝水/睡覺/玩/散步/抱抱）+ 深呼吸 + 番茄鐘 + 4 格天氣。可考慮：
  - 圖示略大、文字略小，或改為僅圖示 + tooltip。
  - 同一區塊內按鈕等寬、圓角一致，間距略增讓「呼吸感」更好。
- **數值條**：compact 時已縮小，可再微調高度或圓角，讓 2×2 數值區更整齊。
- **標題列**：可固定高度、右側按鈕（設定、陪伴模式、白噪音等）對齊方式統一。

### 3. 建議下一步（請選一個方向）

1. **先試用目前緊湊版**：在 400×560 不捲動的前提下看實際使用是否還覺得擠或難點。
2. **若仍擠**：在 B / C / D 中選一個方向（例如「B. 摺疊次要」或「C. 分頁」），再細修要收哪些、放哪一頁。
3. **若主要是「不好看」**：不做大改結構，只做 2. 的視覺微調（間距、圓角、圖示與字級）。

你可以直接回覆：例如「想用 B，把深呼吸和番茄鐘收進一個『更多』」或「想用 C，第一頁只留角色和 6 個動作」，我再依選擇給具體改版方案（含元件與版面配置）。
