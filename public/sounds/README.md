# 自訂音檔使用說明

## 如何添加自訂音檔

### 方法 1: 將音檔放在此資料夾（推薦）

1. 將你的音檔（.mp3, .wav, .ogg 等格式）放到 `public/sounds/` 資料夾
2. 在番茄鐘設定中：
   - 點擊提醒聲或背景音效旁邊的 "Custom" 按鈕
   - 在文字框中輸入音檔路徑，例如：`/sounds/my-alarm.mp3`
   - 按 Enter 確認

### 方法 2: 通過 UI 上傳

1. 在番茄鐘設定中點擊 "Custom" 按鈕
2. 選擇 "選擇檔案" 來上傳你的音檔
3. 音檔會自動載入並使用

## 音檔命名建議

- **提醒聲**: `alarm-gentle.mp3`, `alarm-bell.mp3`, `alarm-chime.mp3`, `alarm-beep.mp3`
- **背景音效**:
  - 滴答聲: `tick.mp3`
  - 火爐聲: `fire.mp3`
  - 下雨聲: `rain.mp3`

## 注意事項

- 支援的音檔格式：MP3, WAV, OGG 等瀏覽器支援的格式
- 如果音檔載入失敗，系統會自動回退到預設的生成音效
- 背景音效會自動循環播放
- 提醒聲只會播放一次
